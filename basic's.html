The Basic's

HTML (HyperText Markup Language)

HTML är en del av webbens 3 grundstenar och är ett märkesspråk som ger strukturen och funktionalitet för webbsidor då dessa går att söka efter ”märkta” innehåll genom taggar såsom exempelvis h1 till h6 vilket ger ett hierarki som underlättar sökoptimering (SEO).

CSS (Cascading Style Sheet)

CSS som namnet antyder så är CSS ett stilmallsformat som kör uppifrån ner och är webbens andra grundsten. Dess funktion är att ”styla” HTML’s märkesspråk (strukturen) som exempelvis allt ifrån hemsidans layout till typsnitt.

JS (JavaScript)

Sist men inte minst har vi JS ett skriptspråk och är den sista av webbens grundstenar och till skillnad från föregående så är JS ett programmeringsspråk där vi kan utföra kommandon och som skapar funktionalitet för hemsidan som komplementeras av strukturen (HTML) och stilen (CSS). Som ett skriptspråk är det enklare att använda samt presterar bättre i hemsidors, som funktioner.
